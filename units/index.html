Css Units

  • Absolute unit:-

    1. 1px

      It takes space of 1 pixel

  • Relative units:-

    1. 1rem

      1x root element's font size

    2. 1em

      1x parent of current element's font size

    3. 1vw

      1% width of the view width

    4. 1vh

      1% height of the view height

    5. 1ch

      1 character width of the character '0' according to the font size
